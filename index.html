O que é HTML ?

HTML é a sigla em inglês para HyperText Matkup Language, ou Linguagem de Marcação de Hipertexto.

Como o próprio significado já diz, ela é uma linguagem de marcação e não de programação.
A Linguagem é responsável por estruturar as páginas web, é basicamente o esqueleto de práticamente todos os sites.

Através das TAGs no HTML é possível adcionar elementos diversos em uma página, seja para um texto, uma imagem, link e etc.




O que são tags ?

As tags são elementos usados para estruturar e formatar o conteúdo em páginas web.

Essas tags são elementos fundamentais do HTML e são usadas para marcar diferentes partes do conteúdo para que o navegador possa interpretá-las corretamente e exibi-las de acordo com a formatação desejada.

Cada tag HTML geralmente possui uma abertura e um fechamento, e o conteúdo é inserido entre essas duas partes. Os sinais usados para marcar a abertura e o fechamento de uma tag são respectivamente os sinais de menor < e maior />, esse segundo com uma barra.

Abaixo as principais tags estudadas nesse módulo e suas utilidades:


Tags de títulos:

As tags de títulos ou também conhecidas como Headings são elementos de nível de blocos que vêm em seis diferentes classificações, do H1 ao H6. Conforme aumenta o valor da tag H o tamanho do título diminui.
Os títulos ajudam a dividir o conteúdo e estabelecer hierarquia, com esses identificadores a leitura das páginas se tornam mais organizada e agradável.

Tags de texto:

Existem inúmeras tags para utilizar em textos.

PARÁGRAFOS:
Os títulos são frequentemente seguidos por parágrafos de apoio, esses parágrafos são definidos usando o elemento de nível de bloco p.
Os parágrafos podem aparecer um após o outro, adicionando textos informativos em páginas web.

NEGRITO, ITÁLICO, SUBLINHADO: Para desatacar palavras ou letras dos textos escritos são usadas algumas tags de formatação de texto.
Existe a tag utilizada para deixar o texto em negrito, usando o elemento strong ;
Existe a tag utilizada para deixar o texto em itálico, usando o elemento i ;
Existe a tag utilizada para deixar o texto sublinhado, usando o elemento u ;

Marcar texto: Para marcar um texto é utilizada a tag mark

Texto pequeno: Para definir textos menores é utilizada a tag small.
Esses textos menores podem ser usados em comentários secundários ou aviso de direitos autorais.

Criando Listas Ordenadas:

Para contruir listas Ordenadas são utilizados os elementos ol e li.

As Listas Ordenadas são organizadas em ordem crescente obedecendo uma ordem numérica.
O elemento ol define que a lista é ordenada, e o elemento li define os itens individuais da lista.

Criando Listas Não Ordenadas:

Para construir listas não ordenadas os elementos utilizados são ul e li

O elemento ul define que a lista é do tipo não ordenada, e o elemento li, igual acontece na lista ordenada, define os itens individuais nesta lista.
Nas listas não ordenadas a ordem em que a lista está construída não importa.
Exemplo de uma lista não ordenada cujo os itens são de um kit de ferramentas usadas por um técnico de redes de computadores:

  • Alicate Grimpador RJ45 e RJ11
  • Testador de Cabo de Rede
  • Desencapador de Fios
  • Alicate Punch Down de Inserção para módulos RJ11 e RJ45
  • Stripper e Cortador de fios multifunções
  • Chave de Fenda
  • Chave Phillips
  • Conectores cristal de RJ45 8P8C
  • Caixa organizadora de PVC com tampa
  • Bateria 9V
  • Bolsa para transporte e proteção

Um recurso que torna as listas extremamente poderosas é a capacidade de serem aninhadas. Cada lista pode ser colocada dentro de outra lista.


Links no HTML:

O elemento a é o elemento âncora, que com o atributo href cria-se uma hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.
O conteúdo dentro de cada tag a precisará indicar o destino do link referente à página desejada.

Exemplo de link: Página inicial do YouTube


Linha Horizontal:

O elemento hr cria uma linha horizontal na página, pode ser usado para dividir diferentes seções e/ou conteúdos do site.


Citações

Para fazer citações no HTML é usado o elemento blockquote
O elemento faz o recuo do texto automaticamente.
